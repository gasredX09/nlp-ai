# transformer_from_scratch

A hands-on PyTorch implementation of the Transformer architecture from scratch.

## Goals
- Understand every major Transformer component by implementing it manually
- Build intuition for tensor shapes, masking, and attention math
- Progress from encoder-decoder Transformer to decoder-only GPT-style models

## Planned modules
- token embeddings
- positional encoding
- scaled dot-product attention
- multi-head attention
- feed-forward network
- encoder block
- decoder block
- full Transformer
- toy training scripts
